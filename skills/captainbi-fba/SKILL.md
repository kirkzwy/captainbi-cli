# CaptainBI FBA

Read `captainbi-shared` first.

- Inventory: `cbi fba inventory`
- Shipments: `cbi fba shipments`
- Storage fee: `cbi fba storage-fee`
- ASIN monitor: `cbi fba asin-monitor`
- Sync shipment: `cbi fba sync-shipment --confirm`
